# Slack Website Clone

A fully responsive front-end recreation of Slack's modern landing page built using semantic HTML5, modular CSS3, and Vanilla JavaScript. This project focuses on advanced DOM manipulation, dynamic UI interactions, accessibility best practices, and responsive design without relying on external frameworks or libraries.

## 🚀 Technologies Used

### HTML5

- Semantic page structure using `header`, `nav`, `main`, `section`, and `footer`
- Accessibility enhancements through ARIA attributes and meaningful document hierarchy
- Responsive content organization for desktop and mobile layouts

### CSS3

- Custom design system using CSS variables
- Flexbox and CSS Grid for responsive layouts
- Modular styling architecture with dedicated media query management
- Smooth transitions, animations, and interactive state styling

### JavaScript (ES6+)

- Event-driven architecture using custom event listeners
- Dynamic DOM manipulation and UI state management
- Interactive tab systems, dropdown menus, modals, and carousels
- HTML5 Video API integration with custom playback controls
- Mobile navigation and responsive interaction handling
